# Electronics Store Management 💻

A Java-based CLI system designed to process, and evaluate data from an Electronics Store simulating its functions.

## 📝 Description

This program simulates an electronisc store terminal. It receives orders from the user, such as creating a product or reducing its quantity and updates the store's/product's information during runtime.
